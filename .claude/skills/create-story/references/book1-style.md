# Modern Chinese 1 (時代華語 1) — style reference

Notes taken directly from the source textbook (`時代華語 1 課程.pdf`, Lesson 1: 新同學 / The New Classmate). Use this to keep generated stories consistent with how this book actually teaches beginners, rather than generic "textbook Chinese."

## Script and register

- **Traditional characters** are the primary script (繁體字), not Simplified. Every generated story must be in Traditional Chinese.
- Dialogue is between named young people who are peers (classmates, friends) — not "Person A / Person B", not adults in formal register.
- Register is casual-polite spoken Mandarin: short sentences, real speech rhythms, no literary or written-style phrasing.

## Lesson 1 dialogue (verbatim example — tone and length to imitate)

```
（在教室 In the classroom）
中明：宜文，她是誰？
宜文：她是新同學，叫友美。她很可愛。
中明：她是哪國人？妳知道嗎？
宜文：我知道，她是日本人。
中明：她很漂亮。
```

Six short lines. Each line is one simple clause or a clause + a short follow-up question. No sentence exceeds ~8 characters excluding the speaker label. This is the *ceiling* for "easy" level — hard level for A1-A2 scope should still look like short spoken exchanges, just with more clauses per turn and slightly more complex connectors.

## Vocabulary annotation format

The book lists new words per lesson as: number, word, pinyin, part-of-speech tag, English gloss, then one example sentence (characters, pinyin, English). Example:

```
1  新    xīn      Vs   new
2  同學  tóngxué  N    classmate
   他是新同學。
   Tā shì xīn tóngxué.
   He is the new classmate.
5  叫    jiào     Vst  to be called
   我叫王宜文。
   Wǒ jiào Wáng Yíwén.
   My name is Wang Yiwen.
```

Common POS tags used: `N` (noun), `V` (action verb), `Vs` (stative verb / adjective-like verb, e.g. 可愛), `Vst` (verb that introduces a state/identity, e.g. 叫/姓), `Adv` (adverb, e.g. 很), `Det` (determiner, e.g. 哪).

## Grammar patterns at this level

Lesson 1 alone establishes the core A1 sentence skeleton that most easy-level output should lean on:
- `X 是 Y` — X is Y (identity/classification)
- `很 + stative verb` — 她很可愛 / 她很漂亮 (no 是 before adjectives)
- `叫 / 姓` — naming: 我叫..., 我姓...
- `哪 + noun` — 哪國人 (which country), asking for a category
- `誰` — who
- Yes/no confirmation tag `...嗎？` and content questions with question words in situ (no inversion)

Medium/hard levels within A1-A2 can add: time words (今天/昨天/明天), location `在`, possession `有`, simple connectors (可是, 因為...所以...), comparison `比`, the experiential/change-of-state markers `了`/`過` — but the *sentence rhythm* should stay short and spoken, matching the source book's voice.
